Le point le plus dur va être de tester le système de nœuds/liens sans éditeur visuel, parce que construire un Blueprint "à la main" en code pour le tester est fastidieux. Une astuce possible : écrire un petit Blueprint de test directement en JSON (vu que tu as déjà le format de sauvegarde défini) et le charger via SauvegardeLogique, plutôt que de le construire objet par objet en Java à chaque test.
